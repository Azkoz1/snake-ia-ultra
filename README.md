# 🐍 Snake AI - Ultra Edition

Un jeu de Snake moderne et minimaliste doté d'une **Intelligence Artificielle** autonome, conçu avec les technologies Web standards (HTML5, CSS3, JavaScript).

![Licence](https://img.shields.io/badge/license-MIT-blue.svg)
![Langage](https://img.shields.io/badge/language-JavaScript-yellow.svg)

## ✨ Caractéristiques

- **IA Autonome :** Utilise un algorithme de recherche de chemin **BFS (Breadth-First Search)** pour trouver la pomme de manière optimale.
- **Mode Survie :** Si l'IA est bloquée, elle active un mode d'évitement pour rester en vie le plus longtemps possible.
- **Design Cyberpunk :** Interface sombre avec effets de lueur néon (Glow) et animations fluides.
- **Mode Manuel :** Basculez à tout moment pour reprendre le contrôle avec les flèches du clavier.
- **Contrôle de Vitesse :** Ajustez la réactivité du jeu en temps réel via un curseur dédié.
